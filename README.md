## What this repo demonstrates

| What | How |
|---|---|
| Reproducibility-aware script | `analysis.R` — qualified `pkg::fn` calls, seeded randomness |
| Risk scoring | `audit_script()` + `risk_score()` — 8 calls detected, 0 risks |
| Output certification | `certify()` — 12 outputs hashed and stored in `.reproducr.rds` |
| Drift detection | `check_drift()` — runs on every CI push against the last certification |
| Auto-updating badge | `repro_badge()` — updated in GitHub Actions on every push |
| Audit report | `reproducibility_report.md` — auto-generated by CI |

## Reproducibility pipeline

### Tier 1 — Scan and score

```r
library(reproducr)
source("analysis.R")

report <- audit_script("analysis.R", verbose = FALSE)
risks  <- risk_score(report)
print(risks)
#>
#> -- reproducr risk score --
#>
#>   No risks detected. All checks passed.
```

### Tier 2 — Certify and check drift

```r
# First run — establish the baseline
certify(OUTPUTS, tag = "baseline-v1", script = "analysis.R")
#> reproducr: certified 12 output(s) [2026-05-31] under tag 'baseline-v1'

# After any environment change — check for drift
check_drift(OUTPUTS, against = "latest")
#>
#> -- reproducr drift check vs 'baseline-v1' --
#>
#>   Verdict  : ALL OUTPUTS MATCH
#>   OK       : 12
#>   Drifted  : 0
```

### Tier 3 — Report and badge

```r
repro_report(report, risks, format = "html", style = "pharma",
             output_file = "qc_report.html")

repro_badge(report, risks, output = "README")
```

## CI/CD

The GitHub Actions workflow (`.github/workflows/reproducr-audit.yml`) runs
on every push to `main` and weekly on Monday:

1. Sources `analysis.R` to produce `OUTPUTS`
2. Audits with `audit_script()` and `risk_score()`
3. Checks for drift against the last certified run
4. Certifies this run with today's date as the tag
5. Updates the badge and generates `reproducibility_report.md`
6. Commits `README.md`, `reproducibility_report.md`, and `.reproducr.rds`

The `.reproducr.rds` file is committed to version control — it is the
auditable history of every certified run.

## Running locally

```r
# Install dependencies
install.packages("palmerpenguins")
install.packages("remotes")
remotes::install_github("ndohpenngit/reproducr")

# Run the full pipeline
source("analysis.R")

library(reproducr)
report <- audit_script("analysis.R", verbose = FALSE)
risks  <- risk_score(report)
print(risks)

certify(OUTPUTS, tag = "local-run")
check_drift(OUTPUTS, against = "local-run")

repro_report(report, risks, format = "html", style = "minimal")
```
